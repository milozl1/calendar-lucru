## üöÄ Deployment

### Quick Start
1. Clone or download this repository
2. Update placeholder content (see below)
3. Upload files to your web hosting
4. Done! No build process required

### Files
- `index.html` - Main website structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive functionality

### Before Going Live

#### 1. Update Contact Information
Replace placeholder values in `index.html`:
- **Phone Number**: Search for `40700000000` and replace with actual number (3 locations)
- **Address**: Update "Strada Frumuse»õii, Nr. 15, Bucure»ôti"
- **Social Media**: Replace `elegancestudio` with actual handles
- **Business Hours**: Adjust opening hours if needed

#### 2. Customize Content
- **Brand Name**: Replace "Elegance Studio" with your salon name
- **Headlines & Copy**: Personalize the story and service descriptions
- **Services**: Add/remove services based on your offerings
- **Testimonials**: Use real client reviews (with permission)

#### 3. Add Images
Replace placeholder image sections:
- **About Section**: Add your photo (`.despre-image`)
- **Hero Background**: Consider adding a subtle beauty/wellness image
- **Contact Map**: Embed Google Maps with your actual location

#### 4. Update Colors (Optional)
Customize the color scheme in `styles.css`:
```css
:root {
    --color-primary: #D4A5A5;      /* Main brand color */
    --color-primary-dark: #B88A8A; /* Darker shade */
    --color-secondary: #F5E6D3;    /* Accent color */
    --color-accent: #E8C4B8;       /* Additional accent */
}
```

#### 5. SEO Optimization
Update meta tags in `<head>`:
- Title tag
- Meta description
- Add Open Graph tags for social media sharing
- Add favicon

## üé® Customization Guide

### Adding New Services
1. Locate the `<div class="servicii-grid">` in `index.html`
2. Copy an existing `<div class="serviciu-card">` block
3. Update icon, title, and description
4. Save and refresh

### Changing Fonts
In `index.html`, replace the Google Fonts link:
```html
<link href="https://fonts.googleapis.com/css2?family=YOUR-FONT&display=swap" rel="stylesheet">
```

Update CSS variables in `styles.css`:
```css
--font-primary: 'Your-Heading-Font', serif;
--font-secondary: 'Your-Body-Font', sans-serif;
```

### Responsive Breakpoints
- Desktop: Default styles
- Tablet: 968px and below
- Mobile: 640px and below
